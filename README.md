Бұл — пайдаланушының қалауына қарай қауіпсіз және күрделі құпия сөздерді генерациялайтын қарапайым бағдарлама. Криптология негіздерін оқу барысында жазылған алғашқы жобаларымның бірі.

🚀 Мүмкіндіктері:
- Құпия сөздің ұзындығын таңдау.
- Бірнеше құпия сөзді бірден жасау.
- Таңдау бойынша келесі таңбаларды қосу:
  - Сандар (`0-9`)
  - Үлкен әріптер (`A-Z`)
  - Кіші әріптер (`a-z`)
  - Арнайы символдар (`!#$%&*+-=?@^_`)
- **Қауіпсіздік:** Оқылуы қиын ұқсас таңбаларды (`il1Lo0O`) алып тастау мүмкіндігі.

 🛠 Технологиялар:
- **Language:** Python 3.x
- **Libraries:** `random`
